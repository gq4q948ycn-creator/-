# Task Generator

**Автор:** [Ваше имя и фамилия]

**Описание:**
Приложение для генерации случайных задач с фильтрацией по типу, сохранением истории и добавлением новых задач. Использует библиотеку `random` для выбора задачи, `json` для сохранения истории и `tkinter` для GUI.

**Функционал:**
- Генерация случайной задачи по нажатию кнопки.
- Отображение истории сгенерированных задач.
- Фильтрация задач по типу (учёба, спорт, работа, отдых).
- Добавление новых задач с указанием типа.
- Сохранение истории в файл `history.json`.
- Проверка на пустой ввод при добавлении задачи.

**Примеры использования:**
1. Нажмите «Сгенерировать задачу» — появится случайная задача.
2. Используйте фильтр, чтобы видеть только задачи определённого типа.
3. Добавьте новую задачу через поле ввода и выберите её тип.
4. История задач сохраняется даже после закрытия приложения.

**Технологии:**
- Python 3.x
- Библиотека `tkinter` (графический интерфейс)
- `random` (генерация случайных задач)
- `json` (сохранение истории)
- `Git` (контроль версий)
